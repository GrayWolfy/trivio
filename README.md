# Trivio

## Клонирование репозитория
```git clone https://github.com/GrayWolfy/trivio.git```

## Сбилдить проект
* ```/trivio```
* ```docker-compose build```
* ```docker-compose up```

## Накатить composer
* ```/trivio/app```
* ```composer init ```
* ```composer install```

## Накатить миграции
* ```docker exec -it trivio_php-fpm_1 bash```
* ```vendor/bin/phinx migrate```

## Описание проекта
Проект реализован с использованием фреймворка Laminas MVC. Он включает в себя следующие функциональности:

## CRUD создания пользователя
Пользователи могут быть созданы, просмотрены, обновлены и удалены из базы данных. Каждый пользователь хранит следующую информацию: ФИО, дата рождения и место рождения. Место рождения представлено в виде списка, заполненного вручную в базе данных.

## Добавление и удаление сотрудников из отдела
Сотрудники могут быть добавлены и удалены из отделов, которые также представлены в виде списка, заполненного вручную в базе данных. Один сотрудник может быть в нескольких отделах.

## Мультиязычность
Сайт поддерживает мультиязычность на двух языках: русском и английском. Переключение языка возможно на любой странице сайта. Выбранный язык сохраняется после перезагрузки страницы.

## Схема базы данных
Схема базы данных описывается следующим образом:

Таблица user

| Колонка   | Тип данных | Описание |
| ------------- | ------------- | -------------|
|id | INT  | идентификатор пользователя |
| name  | VARCHAR(255) | Имя пользователя |
| birthdate  | DATE  | Дата рождения пользователя|
| birthplace_id  | INT  | Идентификатор места рождения пользователя |


Таблица department

| Колонка       | Тип данных | Описание                        |
|---------------| ------------- |---------------------------------|
| id            | INT  | Уникальный идентификатор отдела |
| name_ru       | VARCHAR(255) | Название отдела RU              |
| name_en       | VARCHAR(255)  | Название отдела EN              |

Таблица employee


| Колонка   | Тип данных | Описание                            |
|-----------| ------------- |-------------------------------------|
| id        | INT  | Уникальный идентификатор сотрудника |
| full_name | VARCHAR(255) | Имя Сотрудника                      |


Таблица employee_department


| Колонка   | Тип данных | Описание                        |
|-----------| ------------- |---------------------------------|
| id        | INT  | Уникальный идентификатор |
| employee_id | INT | Id сотрудника                   |
| department_id | INT | Id департамента                 |
